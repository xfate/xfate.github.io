B样条曲线

发表于 2018-08-08 | 分类于 unity ， 数学 | | 本文总阅读量 次

样条是什么

样条本来是指一些细小的有弹性的木条或者钢片，用这些木条来设计模型结构。这样的木条被称之为样条（“spline”）。

在贝塞尔曲线出现之后，人们发现如果要修改曲线的某一部分，该部分的控制点调整会影响到整条曲线。因为了贝塞尔曲线上每个点受所有控制点影响。后来，deBoor 发明了一个曲线，就叫B样条曲线，曲线基于deBoor-Cox递推公式实现。

B样条定义：

这条曲线一共有n+1个控制点Pi。在这条公式里k表示B样条曲线的阶数，k-1表示B样条的次数。注意，B样条和贝塞尔曲线不同的是，贝塞尔曲线的次数和阶数是一样的。为什么t属于[tk-1,tn+1)区间？下面会做解释。

Ni,k(t)是B样条的基函数。

什么是基函数？

基函数 就是一个函数的固定形式，也就是函数只会在这个函数的基础上变化而不会丢掉的函数，在数学中，基函数是函数空间一组特殊的基的元素。对于函数空间中的连续函数都可以表示成一系列基函数的线性组合，就像是在向量空间中每个向量都可以表示成基向量的线性组合一样。

B样条基函数是一个非递减的参数为t的序列所决定的k阶分段多项式。

设t是n + 1个非递减数的集合，t0 <= t1<= t2<= … <= tn。ti称为节点（knots）, 集合t 称为节点向量（knot vector），半开区间[t, ti+1) 是第i个节点区间（knot span）。

如果一个节点 出现 k 次，那么这个节点 是一个重复度（multiplicity）为k 的多重节点，写为 ti(k)。 否则，如果ti只出现一次，它是一个简单节点。如果节点等间距，节点向量或节点序列称为均匀的；否则它是非均匀的。

么产生的曲线不会与控制折线（polyline）的第一边（leg）和最后一边（leg）接触，这种类型的B-样条曲线称为开（open ）B-样条曲线。 我们可能想强制曲线使得它分别与第一个控制点和最后一个控制点的第一边和最后一边相切，像贝塞尔曲线那样。为了做到这些，第一个节点和最后一个节点必须是重复度为k。这就产生了所谓的clamped B-样条曲线。

Unity里实现B样条曲线

1.先实现deboor的递归形式函数：

public static Vector3 DeBoor(int j, int k,int i, float t,List<Vector3> controlPoints,float[] knots)
{
    // 如果j==0，就返回控制点
    if (j == 0)
        return controlPoints[i];
    else
    {
        //pi = （t-ti)/(t_{i+k-j} - ti)
        float param = (t - knots[i]) / (knots[i + k- j] - knots[i]);
        // （1-pi）*deboor（j-1,k,i-1)+ pi*deboor(j-1,k,i)
        return ((1-param) * DeBoor(j - 1, k, i - 1, t, controlPoints, knots)
            + param * DeBoor(j - 1, k, i, t, controlPoints, knots)); 
    }
}

2.根据输入的控制点生成节点knots：

/// <summary>
/// 创建clamped knots
/// </summary>
/// <param name="nControl"></param>
/// <param name="order"></param>
/// <returns></returns>
public static float[] createKnots(int nControl, int order)
{
    // 节点数= 控制点数+次数（或者是阶数+1）n+k
    int nKnots = nControl + order;

    var knots = new float[nKnots];
    for (int i = 0; i < nKnots; i++)
    {
        //除了k-1，n+1 这个区间，其他重复度为k，这是一个clamped B样条。
        if (i < order) 
        {
            knots[i] = 0;
        }

        else if (i < nKnots - order + 1) //n+k-k+1 = n+1
        {
            knots[i] = knots[i - 1] + 1;
        }
        else
        {
            knots[i] = knots[i - 1];
        }
    }
    return knots;
}
